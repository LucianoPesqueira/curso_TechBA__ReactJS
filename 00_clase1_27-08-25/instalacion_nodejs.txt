Crear un proyecto nuevo de React:
-creo una carpeta React e ingreso a la mista
    > npm create vite@latest
    > npx
    > create-vite
    >> Project name: prueba
    >> Select a framework: React
    >> Select a variant: JavaScript
    > cd prueba
    > npm install
    > npm run dev
